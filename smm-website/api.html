API Kalit

API kalitingiz orqali dasturiy tarzda buyurtma berishingiz mumkin.

Baza URL

https://api.seensms.uz/v1

API Endpoints

1. Yangi Buyurtma Yaratish

POST /orders

Headers:
  Authorization: Bearer YOUR_API_KEY
  Content-Type: application/json

Body:
{
  "service": "telegram-members",
  "link": "@username",
  "quantity": 1000
}

Response:
{
  "success": true,
  "order_id": "12345",
  "status": "pending"
}

2. Buyurtma Holati

GET /orders/{order_id}

Headers:
  Authorization: Bearer YOUR_API_KEY

Response:
{
  "order_id": "12345",
  "status": "completed",
  "progress": 100
}

3. Balans

GET /balance

Headers:
  Authorization: Bearer YOUR_API_KEY

Response:
{
  "balance": 100.00,
  "currency": "USD"
}
