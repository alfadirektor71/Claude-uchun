# 🌐 GitHub Pages - Saytni Yoqish

GitHub Pages branch tayyor! Endi faqat yoqish qoldi.

## 🚀 Avtomatik Yo'l (GitHub Actions)

GitHub repository settings da GitHub Pages ni avtomatik yoqish uchun GitHub Actions workflow kerak.

Lekin, siz qo'lda ham yoqishingiz mumkin:

---

## 🔧 Qo'lda Yoqish (1 Daqiqa)

### 1. Repository ga o'ting:
```
https://github.com/alfadirektor71/Claude-uchun
```

### 2. Settings ni oching:
- Repository yuqori qismida **"Settings"** tugmasi
- (Agar ko'rinmasa - sizda admin huquqi yo'q)

### 3. Pages bo'limiga o'ting:
- Chap sidebar da **"Pages"** ni toping
- Yoki to'g'ridan-to'g'ri: https://github.com/alfadirektor71/Claude-uchun/settings/pages

### 4. Source sozlang:
```
Source: Deploy from a branch
Branch: gh-pages
Folder: / (root)
```

**Screenshot:**
```
┌─────────────────────────────────┐
│ Source                          │
│ ○ Deploy from a branch ✓        │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ Branch                          │
│ [gh-pages ▼]  [/ (root) ▼]     │
│              [Save]             │
└─────────────────────────────────┘
```

### 5. Save tugmasini bosing:
- Branch va folder tanlab
- **"Save"** tugmasini bosing
- 1-2 daqiqa kutings

### 6. URL olish:
Deploy tugagach, yuqorida URL ko'rinadi:
```
✅ Your site is live at https://alfadirektor71.github.io/Claude-uchun/
```

## ⚠️ Agar Ochilmasa:

### Sabab 1: Deploy hali tugamagan
- Settings > Pages da deploy status ni tekshiring
- "Your site is published" yozuvi ko'rinishi kerak
- 1-2 daqiqa kutib, refresh qiling

### Sabab 2: Branch noto'g'ri
- gh-pages branch tanlanganligini tekshiring
- Folder: / (root) bo'lishi kerak

### Sabab 3: Permissions
- Repository admin bo'lishingiz kerak
- Yoki collaborator sifatida qo'shilgan bo'lishingiz kerak

## 🎨 Custom Domain (Keyinchalik)

Agar anorsmm.uz ulash kerak bo'lsa:

1. Settings > Pages > Custom domain ga:
   ```
   anorsmm.uz
   ```

2. DNS sozlamalarini qo'shing:
   ```
   Type: A
   Name: @
   Value: 185.199.108.153
   
   Type: CNAME  
   Name: www
   Value: alfadirektor71.github.io
   ```

3. **Enforce HTTPS** ni yoqing
